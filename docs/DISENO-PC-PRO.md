# Alaya Holistics v8.8 · Diseño PC Pro

Esta versión ajusta la web para que se vea mejor en ordenador.

## Mejoras

- Cabecera más estable en escritorio.
- Menú más compacto para que no se amontone.
- Hero más equilibrado.
- Secciones más anchas.
- Tarjetas con mejor distribución.
- Grids optimizados para pantallas grandes.
- Formularios más cómodos.
- Footer más ordenado.
- Panel administrador más ancho.
- Pestañas admin con mejor distribución.

## Importante

La versión mantiene la separación entre cliente y administrador:
- Cliente: `index.html`
- Admin: `admin.html`
