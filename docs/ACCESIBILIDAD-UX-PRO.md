# Alaya Holistics v6.5 · Accesibilidad y UX Pro

Esta versión añade mejoras de accesibilidad y comodidad de lectura.

## Nuevo

- Botón flotante **A11y**.
- Panel de accesibilidad.
- Aumentar texto.
- Reducir texto.
- Modo alto contraste.
- Modo reducir movimiento.
- Restaurar ajustes.
- Preferencias guardadas en localStorage.
- Mejoras de foco visible con teclado.
- Sección pública de accesibilidad.
- Sitemap actualizado con `#accesibilidad-alaya`.

## Objetivo

Que la web sea más cómoda de leer y navegar desde móvil, tablet u ordenador.

## Uso

El usuario puede abrir el botón **A11y** y cambiar:
- tamaño de texto
- contraste visual
- movimiento/animaciones

Los cambios quedan guardados en el navegador.
