# Alaya Holistics v12.6 · Analítica Local y Conversión Pro

Esta versión añade un centro de analítica local en Admin.

## Qué mide

- Visitas locales.
- Vistas de secciones.
- Clics.
- Intención de reserva.
- Intención de contacto.
- Intención de marketing.
- Reservas creadas.
- Clientes CRM.
- Campañas.

## Privacidad

La analítica es local del navegador.
No usa servicios externos.
No debe guardar datos personales.
Puedes limpiar eventos desde Admin.

## Uso recomendado

1. Entrar en Admin → Analítica.
2. Navegar por la web para generar eventos.
3. Revisar embudo.
4. Revisar top secciones.
5. Exportar informe.
6. Usar los datos para mejorar servicios, textos y llamadas a la acción.

## Importante

Si quieres analítica profesional con estadísticas reales multiusuario, haría falta integrar una herramienta externa compatible con privacidad o un backend propio.
