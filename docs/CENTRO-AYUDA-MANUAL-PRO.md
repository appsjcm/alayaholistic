# Alaya Holistics v7.5 · Centro de Ayuda y Manual Pro

Esta versión añade un centro de ayuda público y un manual rápido dentro del admin.

## Nuevo

- Nueva pestaña admin **Ayuda**.
- Sección pública `#ayuda-alaya`.
- Buscador de ayuda.
- Guías por módulo:
  - Reservas
  - Servicios
  - Herbolario
  - Talleres y cursos
  - Cartas astrales admin
  - Backups
  - PWA / instalar app
  - Publicación GitHub Pages
- Vista de guía dentro del admin.
- Botón copiar guía individual.
- Botón copiar manual rápido.
- Botón copiar checklist de formación.
- Botón copiar ayuda para cliente.
- Sitemap actualizado.

## Uso recomendado

- Repasar cómo funciona la web.
- Formar a otra persona que vaya a usar el admin.
- Copiar instrucciones para clientes.
- Tener una guía interna sin salir de la app.

## Nota

El centro de ayuda es orientativo y puede ampliarse en futuras versiones con más guías o tutoriales.
