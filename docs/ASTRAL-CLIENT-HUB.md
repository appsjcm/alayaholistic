# Alaya Holistics v5.1 · Astral Client Hub

Esta versión añade un centro de cliente/paciente dentro del admin astral.

## Nuevo

- Client Hub Astral.
- Resumen automático por cliente/paciente.
- Agrupación por email, teléfono o nombre + fecha + lugar.
- Métricas por cliente:
  - cartas guardadas
  - sesiones
  - sol registrado
  - ascendente registrado
- Último seguimiento visible.
- Botón para filtrar cartas de un cliente.
- Botón para copiar ficha completa.
- Botón para imprimir ficha de cliente.
- Botón para imprimir informe de carta astral.
- Botón para vista previa/imprimir borrador del formulario actual.

## Informes imprimibles

Los informes se abren en una ventana nueva y se pueden imprimir o guardar como PDF desde el navegador.

## Privacidad

Los informes pueden incluir notas privadas. Úsalos solo con consentimiento y cuidado.
