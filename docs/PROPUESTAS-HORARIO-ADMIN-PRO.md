# Alaya Holistics v8.6 · Propuestas de Horario Admin Pro

Esta versión añade una herramienta interna para proponer horarios alternativos.

## Nuevo

- Pestaña admin **Alternativas**.
- Selección de reserva pendiente.
- Hasta 3 opciones de horario alternativo.
- Nota interna opcional.
- Vista previa del mensaje.
- Botón **Proponer horarios y copiar mensaje**.
- Estado **Propuesta alternativa**.
- Botón **Copiar mensaje de espera**.
- Botón **Copiar protocolo**.
- Listado de reservas pendientes o con alternativas.

## Flujo recomendado

1. Cliente solicita reserva.
2. Admin revisa disponibilidad.
3. Si no puede confirmar el horario solicitado, abre **Alternativas**.
4. Añade 2 o 3 opciones.
5. Copia y envía el mensaje al cliente.
6. Cuando el cliente elija, admin confirma manualmente la reserva.

## Importante

La propuesta alternativa no confirma la cita. La cita se confirma solo cuando el administrador confirma manualmente.
