# Alaya Holistics v9.4 · Herbolario Pro

Esta versión mejora la sección Alaya Herbolario.

## Mejoras

- Nueva guía visual antes del catálogo.
- Bloques para:
  - Infusiones y plantas.
  - Aromas y velas.
  - Minerales y detalles.
  - Packs personalizados.
- Recomendación rápida por categoría.
- Formulario de consulta para preparar mensaje.
- Botón para copiar mensaje.
- CTA hacia contacto.
- Mejor lectura en móvil y PC.

## Flujo recomendado para cliente

1. Leer la guía de herbolario.
2. Elegir categoría.
3. Explorar catálogo.
4. Preparar consulta si tiene dudas.
5. Contactar con Alaya.
