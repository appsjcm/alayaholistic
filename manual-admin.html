Gestión interna de Alaya

Esta guía es solo para administración. No debe mezclarse con la ayuda pública de clientes.

01

Revisa solicitudes, usa el centro de confirmaciones, cambia estados, controla bloqueos, calendario y mensajes de confirmación.

02

Servicios

Edita servicios, categorías, duración, precio, descripción, etiquetas y fichas públicas.

03

Herbolario

Gestiona productos, categorías, stock visible, destacados, descripciones y consultas.

04

Talleres

Crea eventos, plazas, fecha, nivel, temario, material, formato y estado.

05

Cartas astrales

Gestiona fichas privadas, consentimiento, notas internas, seguimiento e informes.

06

Backups

Exporta copia completa antes de cambios importantes. Guarda el JSON en un lugar seguro.

07

SEO, calidad y publicación

Revisa metadatos, auditoría de calidad, PWA, GitHub Pages y estado de lanzamiento.

08

Separación cliente/admin

El cliente nunca debe ver notas privadas, backups, métricas, publicación, logs, accesos admin ni herramientas técnicas en la navegación pública.

Entrar al panel interno

Usa el acceso administrador para gestionar la web.

Abrir panel

Admin Visible Pro

La página principal incluye un botón visible de Admin, pero el panel sigue separado en admin.html.

Recomendación: usar este acceso solo para administradores y mantener credenciales seguras.

Panel Admin Pro

El panel incluye un centro rápido con accesos a confirmaciones, alternativas, reservas, calendario, servicios y manual interno.

La rutina recomendada es revisar pendientes, confirmar o proponer alternativa, enviar mensaje al cliente y realizar copia de seguridad si hay cambios importantes.

Rutina de revisión por secciones

  1. Revisar reservas pendientes.
  2. Confirmar, cancelar o proponer alternativas.
  3. Actualizar talleres, plazas y servicios visibles.
  4. Revisar mensajes de contacto o consultas de herbolario.
  5. Comprobar que la web pública no muestra herramientas internas salvo el acceso Admin.
  6. Hacer copia de seguridad tras cambios importantes.

Gestor de Contenido Admin Pro

La pestaña “Gestor contenido” permite crear servicios, talleres, cursos, productos de herbolario, avisos y novedades desde el panel.

  1. Entrar al panel Admin.
  2. Abrir “Gestor contenido”.
  3. Elegir tipo de contenido.
  4. Rellenar título, categoría, estado, fecha, precio y descripción.
  5. Marcar “Publicar visible en la web” si debe verse para clientes.
  6. Guardar contenido.

En modo estático se guarda en el navegador. Para compartir entre dispositivos hace falta activar sincronización online.

Admin WordPress CMS Pro

La pestaña “WordPress CMS” permite gestionar la web como un mini WordPress estático.

  1. Entrar al panel Admin.
  2. Abrir “WordPress CMS”.
  3. Editar identidad, portada, colores, menú y SEO.
  4. Crear bloques públicos desde “Bloques”.
  5. Guardar cambios.
  6. Exportar JSON como copia de seguridad.

En hosting estático los cambios se guardan en el navegador. Para compartir entre dispositivos, conectar Firebase/online sync.

Constructor Visual Admin Pro

La pestaña “Constructor visual” permite crear secciones por plantilla, ordenar bloques y guardar revisiones.

  1. Entrar al panel Admin.
  2. Abrir “Constructor visual”.
  3. Elegir una plantilla rápida.
  4. Editar el bloque desde “WordPress CMS → Bloques”.
  5. Ordenar bloques con subir/bajar.
  6. Guardar una revisión antes de publicar cambios grandes.
  7. Exportar backup JSON si vas a subir una versión nueva al hosting.

Publicador CMS Hosting Pro

La pestaña “Publicador CMS” genera el archivo data/cms-public.json con la configuración pública del CMS.

  1. Editar la web desde Admin.
  2. Entrar en “Publicador CMS”.
  3. Descargar cms-public.json.
  4. Subirlo al hosting dentro de la carpeta data/.
  5. Comprobar la web en incógnito.

Para publicación automática real hace falta Firebase o backend.

CMS Online Sync Pro

La pestaña “Sync CMS” prepara sincronización entre dispositivos y administradores.

  1. Editar contenido desde Admin.
  2. Guardar un snapshot local antes de cambios grandes.
  3. Usar Publicador CMS para generar data/cms-public.json.
  4. Para multiadmin real, configurar Firebase y reglas de seguridad.
  5. Probar subida/bajada del CMS desde Sync CMS.

Sin Firebase, el contenido sigue guardándose en el navegador o en archivo publicado JSON.

Firebase Setup Wizard Pro

La pestaña “Firebase Wizard” ayuda a generar configuración, reglas y diagnóstico para online sync real.

  1. Crear proyecto Firebase.
  2. Activar Authentication y Firestore.
  3. Pegar configuración web en el asistente.
  4. Añadir emails admin autorizados.
  5. Copiar reglas Firestore y revisarlas.
  6. Descargar backend-config.js y subirlo al hosting.
  7. Probar Sync CMS.

Reservas Online Firebase Pro

La pestaña “Reservas online” permite gestionar solicitudes recibidas desde la web pública.

  1. El cliente envía una solicitud desde “Reserva online”.
  2. El sistema genera un código.
  3. Admin revisa la solicitud.
  4. Admin cambia estado: pendiente, confirmada, alternativa, cancelada o completada.
  5. Admin copia el mensaje para responder al cliente.
  6. Si Firebase está configurado, puede sincronizar solicitudes.

La cita no queda confirmada automáticamente. Siempre requiere validación manual de Alaya.

Seguimiento de Reservas Pro

Permite que el cliente consulte el estado con código y contacto. Admin puede guardar respuesta visible, nota interna y propuesta de horario.

  1. Cliente envía una reserva online.
  2. Admin revisa la solicitud.
  3. Admin añade propuesta o respuesta visible.
  4. Cliente consulta estado con código/contacto.
  5. La cita solo queda confirmada cuando Admin cambia el estado a confirmada.

Agenda y Disponibilidad Online Pro

Permite crear horarios disponibles y bloqueos desde Admin.

  1. Entrar en Admin → Agenda online.
  2. Crear un horario con servicio, fecha y hora.
  3. Publicarlo como disponible o plazas limitadas.
  4. El cliente lo verá en Disponibilidad orientativa.
  5. El cliente puede solicitar ese horario desde Reserva online.
  6. Admin confirma manualmente la solicitud.

Mensajes y Notificaciones Pro

La pestaña “Mensajes” permite crear respuestas rápidas para clientes de reservas.

  1. Selecciona una reserva.
  2. Elige una plantilla.
  3. Revisa el mensaje generado.
  4. Copia el texto o abre WhatsApp/email.
  5. Envía manualmente el mensaje al cliente.

En hosting estático no se envían mensajes automáticos.

Clientes CRM e Historial Pro

La pestaña “Clientes” permite gestionar clientes/consultantes, etiquetas, preferencias y notas internas.

  1. Crear cliente manualmente o actualizar desde reservas.
  2. Añadir etiquetas y preferencia general.
  3. Consultar historial de reservas vinculadas.
  4. Copiar resumen si se necesita.
  5. Exportar clientes JSON como copia.

No guardar datos sensibles ni información privada innecesaria.

Dashboard Negocio Pro

La pestaña “Dashboard Pro” reúne métricas, tareas pendientes, agenda, clientes y estado técnico.

  1. Entrar en Admin → Dashboard Pro.
  2. Revisar KPIs principales.
  3. Atender tareas pendientes.
  4. Revisar próximos horarios y clientes recientes.
  5. Exportar informe JSON si se necesita copia.

Backup y Restauración Pro

La pestaña “Backup” permite exportar y restaurar todos los datos internos del Admin.

  1. Entrar en Admin → Backup.
  2. Descargar backup total antes de actualizar.
  3. Guardar el archivo en un lugar seguro.
  4. Para restaurar, pegar el JSON en la zona de importación.
  5. Elegir combinar o reemplazar.
  6. Revisar reservas, clientes, agenda y CMS.

Los backups pueden contener datos de clientes; no compartir públicamente.

Legal, Privacidad y Consentimientos Pro

La pestaña “Legal Pro” permite editar textos de privacidad, consentimiento y avisos de confianza.

  1. Entrar en Admin → Legal Pro.
  2. Completar nombre comercial y contacto.
  3. Revisar consentimiento de reservas.
  4. Revisar aviso de cookies y retención de datos.
  5. Revisar auditoría de privacidad.
  6. Exportar configuración legal como copia.

Es una base orientativa. Revisar antes de publicar con datos reales.

Centro Lanzamiento y QA Pro

La pestaña “Lanzamiento QA” permite revisar la web antes de publicarla.

  1. Descargar backup total.
  2. Revisar Legal Pro.
  3. Probar reserva y seguimiento.
  4. Crear disponibilidad de agenda.
  5. Ejecutar QA.
  6. Resolver tareas pendientes.
  7. Exportar reporte.
  8. Activar modo lanzamiento.

SEO, Rendimiento y Accesibilidad Pro

La pestaña “SEO UX” revisa SEO técnico, PWA, formularios, enlaces y accesibilidad.

  1. Entrar en Admin → SEO UX.
  2. Ejecutar auditoría.
  3. Revisar sugerencias.
  4. Activar lectura cómoda o reducir animaciones si se desea.
  5. Exportar informe.

Release Candidate Final Pro

La pestaña “Release Final” reúne el cierre final de la web: informe global, checklist, módulos incluidos y prioridades.

  1. Descargar backup total.
  2. Ejecutar Lanzamiento QA.
  3. Ejecutar SEO UX.
  4. Abrir Release Final.
  5. Revisar puntuación final.
  6. Exportar paquete final JSON.
  7. Guardar el ZIP v12.0 como versión candidata final.

Publicación Guiada y Dominio Pro

La pestaña “Publicación” prepara hosting, dominio, robots, sitemap y plan de subida.

  1. Elegir plataforma: GitHub Pages, Netlify, Vercel o manual.
  2. Escribir URL pública o dominio.
  3. Generar plan de publicación.
  4. Copiar pasos o exportar JSON.
  5. Copiar robots/sitemap si se van a subir personalizados.
  6. Subir todos los archivos del ZIP descomprimido.

Seguridad Producción y Firebase Rules Pro

La pestaña “Seguridad Pro” ayuda a preparar admins, dominio y reglas Firestore orientativas.

  1. Entrar en Admin → Seguridad Pro.
  2. Configurar modo staging o producción.
  3. Añadir emails admin autorizados.
  4. Añadir dominio permitido.
  5. Copiar reglas Firestore orientativas.
  6. Revisar y probar reglas en Firebase.
  7. Exportar plan de seguridad.

No usar datos reales con reglas abiertas.

Primer Arranque y Datos Demo Pro

La pestaña “Primer Arranque” ayuda a configurar la web y probarla sin datos reales.

  1. Completar datos básicos.
  2. Cargar datos demo.
  3. Probar reservas, agenda, clientes y mensajes.
  4. Borrar datos demo.
  5. Exportar plan inicial.
  6. Descargar backup antes de usar datos reales.

Mantenimiento, Soporte e Incidencias Pro

La pestaña “Mantenimiento” sirve para registrar incidencias, tareas y estado público tras el lanzamiento.

  1. Definir estado público.
  2. Registrar incidencias.
  3. Priorizar tareas.
  4. Cerrar incidencias resueltas.
  5. Exportar reporte antes de cambios grandes.

Marketing, Redes y Difusión Pro

La pestaña “Marketing” sirve para preparar campañas, textos y calendario de difusión.

  1. Crear una campaña.
  2. Elegir canal y objetivo.
  3. Generar texto.
  4. Copiar y revisar el texto.
  5. Publicar manualmente.
  6. Exportar plan de marketing.

Analítica Local y Conversión Pro

La pestaña “Analítica” permite revisar eventos locales, embudo y conversión sin servicios externos.

  1. Navegar por la web para generar eventos.
  2. Abrir Admin → Analítica.
  3. Revisar métricas y embudo.
  4. Ver top secciones.
  5. Exportar informe JSON.
  6. Limpiar eventos locales cuando sea necesario.

Multiidioma y Traducciones Pro

La pestaña “Idiomas” permite editar textos en castellano, catalán e inglés.

  1. Elegir idioma.
  2. Editar textos principales.
  3. Guardar traducciones.
  4. Aplicar idioma.
  5. Exportar diccionario JSON como copia.

Recordatorios y Seguimiento Pro

La pestaña “Recordatorios” ayuda a preparar mensajes manuales de reservas y agenda.

  1. Guardar nombre visible y firma.
  2. Pulsar “Generar recordatorios”.
  3. Copiar el mensaje necesario.
  4. Marcar como preparado, enviado u omitido.
  5. Exportar reporte de seguimiento.

Tarifas, Packs y Servicios Pro

La pestaña “Tarifas” permite gestionar catálogo, precios orientativos, packs y visibilidad pública.

  1. Crear o editar servicio.
  2. Definir categoría, modalidad, duración y precio.
  3. Añadir nota responsable.
  4. Marcar visible u oculto.
  5. Exportar catálogo JSON.

Talleres, Cursos e Inscripciones Pro

La pestaña “Talleres” permite crear actividades, controlar plazas y gestionar solicitudes.

  1. Crear taller o curso.
  2. Definir fecha, hora, duración, plazas y precio orientativo.
  3. Marcar visible para publicarlo.
  4. Revisar solicitudes recibidas.
  5. Marcar como contactada, confirmada, lista de espera o cancelada.
  6. Exportar listado JSON.
