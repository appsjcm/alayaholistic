# Alaya Holistics v5.9 · Contacto y Conversión Pro

Esta versión mejora la parte pública enfocada a contacto y reservas.

## Nuevo

- Sección pública **Contacto y reservas**.
- Tarjetas:
  - Reserva con cita previa
  - WhatsApp
  - Horario orientativo
- Constructor de mensaje rápido.
- Botón para copiar mensaje de contacto.
- Vista previa del mensaje.
- Botón flotante **volver arriba**.
- Ribbon en hero: reservas con cita previa / horas concertadas.
- Acceso rápido actualizado hacia contacto.
- Sitemap actualizado con `#contacto-alaya`.

## Objetivo

Que el cliente entienda rápidamente:
- qué puede hacer
- cómo contactar
- cómo reservar
- qué mensaje enviar

## Recomendación

Cuando se configure el WhatsApp real, se puede convertir el mensaje rápido en enlace directo a WhatsApp con texto precargado.
