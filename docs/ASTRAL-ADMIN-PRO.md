# Alaya Holistics v4.9 · Astral Admin Pro

Esta versión mueve Alaya Astral IA al panel administrador y añade gestión de cartas astrales.

## Cambios principales

- La app Astral IA deja de mostrarse como acceso público.
- El acceso a `alaya-astral/index.html` queda enlazado desde el panel admin.
- Nueva pestaña admin: **Cartas astrales**.
- Guardado local de cartas astrales en `localStorage`.
- Fichas con datos de cliente/paciente.
- Notas privadas y seguimiento interno.
- Exportación JSON y CSV.
- Búsqueda y filtro por estado.

## Campos de carta astral

- Nombre
- Código interno
- Email

- Fecha de nacimiento
- Hora de nacimiento
- Lugar
- País
- Zona horaria
- Estado
- Tema de consulta
- Sol
- Luna
- Ascendente
- Medio Cielo
- Posiciones planetarias
- Casas
- Aspectos
- Interpretación
- Notas privadas
- Seguimiento
- Consentimiento

## Privacidad

El formulario incluye consentimiento para guardar datos de carta astral y notas internas.

Recomendación: si la web se usa en producción, revisar protección de datos y privacidad con asesoramiento profesional.
