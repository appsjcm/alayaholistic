# Alaya Holistics v12.8 · Recordatorios y Seguimiento Pro

Esta versión añade un centro de seguimiento manual en Admin.

## Qué permite

- Generar recordatorios desde reservas y agenda.
- Preparar mensajes listos para copiar.
- Marcar recordatorios como:
  - pendiente,
  - preparado,
  - enviado manual,
  - omitido.
- Filtrar y buscar recordatorios.
- Exportar reporte JSON.
- Copiar resumen.

## Tipos de recordatorio

- Reserva pendiente.
- Recordatorio 24h.
- Seguimiento posterior.
- Propuesta de horario.
- Disponibilidad de agenda.

## Importante

El sistema no envía mensajes automáticamente.
Solo prepara textos para copiar en WhatsApp, email u otro canal.
