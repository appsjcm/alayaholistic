# Alaya Holistics v12.5 · Marketing, Redes y Difusión Pro

Esta versión añade un centro de difusión en Admin.

## Qué permite

- Crear campañas.
- Planificar fecha y canal.
- Generar textos para:
  - Instagram,
  - WhatsApp,
  - email/newsletter,
  - Google Business,
  - anuncio corto.
- Guardar campañas.
- Ver calendario.
- Copiar textos.
- Exportar plan JSON.
- Consultar ideas rápidas.

## Uso recomendado

1. Entrar en Admin → Marketing.
2. Crear campaña.
3. Elegir canal.
4. Generar texto.
5. Revisar y adaptar.
6. Copiar texto.
7. Publicarlo manualmente en la red/canal elegido.
8. Marcar campaña como finalizada.

## Privacidad

No añadir datos privados de clientes.
El sistema solo prepara textos; no envía mensajes automáticamente.
