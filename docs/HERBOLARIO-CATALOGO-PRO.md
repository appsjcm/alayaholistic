# Alaya Holistics v4.1 · Herbolario Catálogo Pro

Esta versión mejora el catálogo de Alaya Herbolario.

## Nuevo en la parte pública

- Producto destacado.
- Filtro por stock.
- Botón limpiar filtros.
- Etiquetas clicables.
- Ficha ampliada de producto.
- Modal con información completa.
- Consulta por WhatsApp desde la ficha.
- Información visible de categoría, stock, formato, uso, ingredientes y notas.

## Nuevo en el panel admin

Los productos ahora pueden tener:

- categoría
- stock
- icono
- formato
- etiquetas
- producto destacado
- descripción
- uso recomendado
- ingredientes o contenido
- notas de seguridad/conservación/consulta

## Importante

Los textos están pensados para catálogo informativo y consulta previa. No se añade pago online todavía.
