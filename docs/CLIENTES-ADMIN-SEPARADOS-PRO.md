# Alaya Holistics v7.6 · Clientes y Admin Separados Pro

Esta versión corrige la mezcla entre contenido para clientes y contenido interno de administración.

## Cambios principales

- La web pública queda enfocada a **clientes**.
- El panel admin queda enfocado a **administrador**.
- Se añade bloque visual **Zona clientes**.
- Se añade bloque visual **Zona administrador**.
- La ayuda pública muestra solo guías para clientes.
- La ayuda admin muestra solo guías internas.
- Se ocultan de la navegación pública:
  - Backups
  - Publicación
  - Calidad técnica
  - Versión interna
  - Métricas técnicas
- Se añade centro interno admin con accesos a:
  - Backups
  - Calidad
  - SEO
  - Lanzamiento
  - Mantenimiento
  - Métricas
- Se añade checklist de separación cliente/admin.

## Qué debe ver un cliente

- Servicios
- Reservas
- Herbolario
- Talleres
- Astrología pública
- Asistente
- Buscador
- Novedades
- Opiniones / FAQ
- Contacto
- Ayuda cliente
- Instalar app

## Qué debe quedar solo para administrador

- Reservas internas y estados
- Agenda
- Bloqueos
- Productos / servicios / talleres editables
- Cartas astrales y notas privadas
- Backups
- Métricas
- SEO
- Calidad técnica
- Publicación GitHub Pages
- Mantenimiento
- Logs / auditoría
- Firebase / sincronización
