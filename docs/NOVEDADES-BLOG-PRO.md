# Alaya Holistics v6.2 · Novedades y Blog Pro

Esta versión añade una sección pública de novedades y un editor admin para gestionarlas.

## Nuevo

- Nueva sección pública **Novedades Alaya**.
- Tarjeta destacada.
- Grid de artículos/novedades.
- Filtros por categoría:
  - Novedad
  - Taller
  - Herbolario
  - Astrología
  - Bienestar
- Modal de lectura completa.
- Botón para copiar resumen.
- Editor admin dentro de **Contenido**:
  - título
  - categoría
  - fecha
  - estado publicado/borrador
  - icono/emoji
  - resumen
  - contenido completo
  - destacar en web pública
- CRUD de novedades:
  - crear
  - editar
  - borrar
  - cargar ejemplos
- Sitemap actualizado con `#novedades-alaya`.

## Uso recomendado

1. Entrar en admin.
2. Abrir la pestaña **Contenido**.
3. Crear novedades o artículos.
4. Marcar como **Publicado** para que aparezcan en la web.
5. Usar **Borrador** para preparar contenido antes de publicarlo.

## Nota

Los textos deben mantenerse como orientación, bienestar y autoconocimiento. Evitar promesas absolutas o mensajes médicos.
