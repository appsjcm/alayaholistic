# Alaya Holistics v11.3 · Mensajes y Notificaciones Pro

Esta versión añade un centro de mensajes en Admin.

## Qué permite

- Seleccionar una reserva.
- Elegir plantilla.
- Generar mensaje con datos reales.
- Editar el mensaje.
- Copiar texto.
- Abrir WhatsApp.
- Abrir email.
- Ver avisos internos.
- Exportar plantillas y registro local.

## Plantillas

- Reserva recibida.
- Reserva confirmada.
- Propuesta alternativa.
- Recordatorio.
- Cancelación.
- Agradecimiento.
- Personalizado.

## Importante

En hosting estático, la web no envía mensajes automáticamente.
El Admin prepara el texto y abre WhatsApp/email para enviarlo manualmente.

Para automatizaciones reales de email o SMS hace falta backend o servicio externo seguro.
