# Alaya Holistics v7.3 · Avisos Globales Pro

Esta versión añade un sistema de avisos públicos configurables desde admin.

## Nuevo

- Nueva pestaña admin **Avisos**.
- Activar/desactivar aviso público.
- Tipos de aviso:
  - Normal
  - Importante
  - Taller
  - Herbolario
  - Astrología
- Modos:
  - Banner + tarjeta
  - Solo banner
  - Solo tarjeta
- Título configurable.
- Mensaje configurable.
- Botón CTA configurable.
- Enlace configurable.
- Opción para permitir cerrar el banner.
- Vista previa en admin.
- Banner público superior.
- Tarjeta pública en sección `#avisos-alaya`.
- Botón copiar aviso.
- Botón copiar checklist.
- Sitemap actualizado.

## Uso recomendado

Usar para:

- Cambios de horario.
- Vacaciones.
- Talleres destacados.
- Nueva consulta de carta astral.
- Novedades del herbolario.
- Mensajes importantes sobre reservas.
- Campañas temporales.

## Buenas prácticas

- Mantener el mensaje breve.
- Revisar el enlace del botón.
- Desactivar el aviso cuando deje de ser actual.
- Evitar promesas médicas o absolutas.
