# Alaya Holistics v3.0 · Calendario Pro

Esta versión añade una vista mensual para administrar reservas.

## Nuevo

- Pestaña **Calendario** en el panel admin.
- Vista mensual de reservas.
- Indicador de citas por día.
- Colores por estado:
  - pendiente
  - confirmada
  - cancelada
- Botón hoy.
- Navegación mes anterior / siguiente.
- Resumen del día seleccionado.
- Listado de reservas del día.
- Acciones rápidas:
  - contactar
  - copiar confirmación
  - copiar recordatorio
  - imprimir

## Para qué sirve

Permite ver la agenda de un vistazo y no depender solo de la lista de reservas.

## Importante

La vista calendario usa las reservas cargadas en la web.

Si usas Firebase, primero carga/sincroniza datos online para ver todas las reservas en el calendario.
