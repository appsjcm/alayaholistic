# Alaya Holistics v7.1 · SEO y Compartir Pro

Esta versión añade herramientas para preparar SEO y textos para compartir.

## Nuevo

- Nueva pestaña admin **SEO**.
- Campos:
  - título SEO
  - descripción SEO
  - keywords
  - URL pública
  - imagen social
- Vista previa social.
- Generador de meta tags HTML.
- Generador de texto para compartir.
- Botón copiar meta HTML.
- Botón copiar texto compartir.
- Botón copiar checklist SEO.
- Sección pública **Compartir Alaya**.
- Botones públicos:
  - copiar enlace
  - compartir con API nativa si el dispositivo lo permite
- Sitemap actualizado con `#compartir-alaya`.

## Nota importante

Los cambios guardados en el panel son locales del navegador. Para SEO real fijo en buscadores,
conviene copiar las meta tags generadas y pegarlas en el `<head>` final antes de publicar.

## Uso recomendado

1. Entrar en admin.
2. Abrir pestaña **SEO**.
3. Ajustar título, descripción, URL e imagen.
4. Copiar meta tags.
5. Pegarlas en `index.html` antes de publicar si se quiere SEO fijo.
6. Probar al compartir en WhatsApp o redes.
