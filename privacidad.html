Política de privacidad

Texto base orientativo. Antes de publicar una web real, conviene revisar estos textos con asesoramiento profesional y adaptarlos a los datos reales del negocio.

1. Responsable

Responsable pendiente de completar con los datos reales de Alaya antes de publicación.

2. Datos que puede recoger la web

  • Nombre.
  • Email o teléfono.
  • Servicio solicitado.
  • Fecha y hora de reserva.
  • Mensaje enviado por el formulario.
  • Datos natales y notas internas en el área admin, solo si se usan cartas astrales.

3. Finalidad

Gestionar reservas, responder solicitudes, organizar talleres, administrar servicios y, si procede, llevar seguimiento interno de consultas astrológicas con consentimiento.

4. Base de legitimación

La base puede ser el consentimiento del usuario y/o la solicitud de servicios. Debe revisarse según el uso real de la web.

5. Conservación

Los datos deben conservarse solo durante el tiempo necesario para gestionar la solicitud o relación con el cliente. La web incluye campos internos para revisión de datos y consentimiento.

6. Derechos

El usuario puede solicitar acceso, rectificación o eliminación de sus datos. Añadir aquí el contacto oficial para ejercer derechos.

7. Seguridad

El área admin debe protegerse con acceso seguro. Si se usa Firebase, las reglas deben limitar el acceso a administradores autorizados.

8. Menores

Los servicios y formularios deben usarse con autorización adecuada cuando corresponda.
