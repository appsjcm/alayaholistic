# Alaya Holistics v12.9 · Tarifas, Packs y Servicios Pro

Esta versión añade un catálogo de servicios y packs.

## Qué permite

- Crear servicios.
- Editar precios orientativos.
- Definir duración.
- Definir modalidad.
- Marcar disponibilidad.
- Mostrar u ocultar en la web pública.
- Copiar ficha de servicio.
- Exportar catálogo JSON.
- Restaurar catálogo base.

## Categorías

- Tarot / cartas.
- Reiki.
- Holístico.
- Herbolario.
- Talleres.
- Packs.

## Importante

Los precios son orientativos y deben confirmarse manualmente.
Los textos deben ser responsables: no prometer curas, diagnósticos ni resultados médicos.
