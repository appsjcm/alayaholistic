# Alaya Holistics v5.5 · Informes Astrales Pro

Esta versión separa informes internos de informes para entregar al cliente/paciente.

## Nuevo

- Panel **Informes Astrales Pro**.
- Campos específicos para informe cliente:
  - título del informe
  - resumen inicial
  - mensaje final
- Botón **Plantilla informe cliente**.
- Botón **Vista informe cliente**.
- Botón **Informe cliente** en cada carta.
- Botón **Informe interno** renombrado.
- Checklist de informe copiable.
- Informe cliente imprimible / guardable como PDF desde navegador.
- Informe cliente no incluye notas privadas ni seguimiento interno por defecto.

## Diferencia entre informes

### Informe cliente
Pensado para entregar. Incluye:
- datos de carta
- claves principales
- resumen
- interpretación
- fortalezas
- retos
- recomendaciones
- mensaje final

### Informe interno
Pensado para Alaya admin. Puede incluir:
- notas privadas
- seguimiento interno
- sesiones
- próximos pasos

## Recomendación

Antes de entregar cualquier informe, revisar que no incluya datos o notas privadas innecesarias.
