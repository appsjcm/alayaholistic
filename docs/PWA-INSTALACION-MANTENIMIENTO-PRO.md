# Alaya Holistics v6.7 · PWA Instalación y Mantenimiento Pro

Esta versión mejora la experiencia como app instalable.

## Nuevo

- Nueva sección pública **Alaya como app**.
- Botón principal **Instalar app**.
- Botón para copiar instrucciones.
- Botón flotante de instalación cuando el navegador lo permite.
- Estado público de instalación.
- Panel PWA dentro de **Lanzamiento**.
- Estado de:
  - manifest
  - Service Worker
  - modo instalación
  - cachés
- Botón **Comprobar PWA**.
- Botón **Limpiar caché PWA**.
- Botón **Copiar guía instalación**.
- Manifest actualizado con shortcuts:
  - Reservar
  - Servicios
  - Herbolario
  - Contacto
- Sitemap actualizado con `#instalar-alaya`.

## Uso recomendado

1. Publicar la web en HTTPS.
2. Abrir desde móvil.
3. Probar instalar como app.
4. Si no aparecen cambios tras actualizar archivos, usar **Limpiar caché PWA**.
5. Probar `offline.html`.

## Nota

La instalación PWA depende del navegador y del sistema operativo.
En iPhone normalmente se usa Safari → Compartir → Añadir a pantalla de inicio.
