# Alaya Holistics v4.6 · Alaya Astral IA integrada

Esta versión integra la app **Alaya Astral IA** dentro de la web principal.

## Carpeta añadida

```txt
alaya-astral/
```

La app astral se abre desde:

```txt
alaya-astral/index.html
```

## Qué se añade a la web principal

- Nueva sección pública **Alaya Astral IA**.
- Botón **Abrir Alaya Astral IA**.
- Botón **Reservar consulta astral**.
- Logo de Alaya Astrología integrado en la sección.
- Panel admin con acceso directo a la app astral.
- Documentación de integración.
