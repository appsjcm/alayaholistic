# Alaya Holistics v11.5 · Dashboard Negocio Pro

Esta versión añade un panel de informes en Admin.

## Qué muestra

- Reservas totales.
- Reservas pendientes.
- Reservas confirmadas.
- Clientes CRM.
- Horarios visibles.
- Mensajes preparados.
- Tareas pendientes.
- Reservas por estado.
- Reservas por servicio.
- Próximos horarios.
- Clientes recientes.
- Estado técnico.

## Acciones

- Exportar informe JSON.
- Copiar resumen ejecutivo.

## Uso recomendado

1. Entrar en Admin.
2. Abrir “Dashboard Pro”.
3. Revisar tareas pendientes.
4. Revisar reservas pendientes.
5. Publicar disponibilidad si hace falta.
6. Exportar informe como copia.

## Nota

Las métricas se calculan con datos locales o sincronizados.
Para visión multiadmin real, configurar Firebase/online sync.
