# Alaya Holistics v4.0 · Alaya Herbolario Pro

Esta versión separa el herbolario como una sección propia dentro de la web.

## Qué cambia

Antes el herbolario estaba integrado como una parte pequeña de Alaya Holistics.

Ahora tiene una zona propia:

```txt
Alaya Herbolario
```

con catálogo, filtros, buscador y fichas más completas.

## Nuevo en la parte pública

- Sección independiente **Alaya Herbolario**.
- Hero propio del herbolario.
- Botón directo para consultar por WhatsApp.
- Beneficios visuales:
  - natural
  - energético
  - cercano
- Buscador de productos.
- Filtros por categoría.
- Contadores destacados.
- Fichas de producto más completas.

## Categorías

- Infusiones
- Plantas
- Aromas
- Minerales
- Velas
- Packs
- Complementos

## Nuevo en el panel admin

En:

```txt
Panel admin > Herbolario / productos
```

ahora puedes editar:

- nombre
- precio
- categoría
- stock
- icono
- etiquetas
- descripción
- uso recomendado o consejo

## Importante

Los textos del herbolario se han orientado a bienestar, autocuidado y consulta personalizada.

Se evita prometer curaciones o efectos médicos.
