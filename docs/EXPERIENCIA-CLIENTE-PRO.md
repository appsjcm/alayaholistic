# Alaya Holistics v8.0 · Experiencia Cliente Pro

Esta versión mejora la experiencia pública del cliente sin mezclar herramientas de administrador.

## Nuevo

- Sección **Qué puedes hacer en Alaya Holistics**.
- Sección **Elige tu camino**.
- Sección **Cómo funciona una reserva**.
- Mini guía antes de reservar.
- Accesos rápidos por intención:
  - no sé qué elegir
  - quiero una consulta
  - carta astral
  - productos
  - talleres
  - contacto
- Enlaces limpios para clientes.
- Sitemap actualizado.

## Objetivo

Que la persona cliente entienda rápidamente:
1. qué puede hacer en la web,
2. cómo reservar,
3. dónde consultar productos,
4. cómo contactar,
5. qué camino elegir si tiene dudas.

## Nota

Las herramientas internas siguen separadas en `admin.html` y no deben aparecer en la experiencia pública.
