# Alaya Holistics v9.3 · Servicios Pro

Esta versión mejora la sección de servicios.

## Mejoras

- Nueva guía visual antes de servicios.
- Tarjetas para:
  - Lecturas.
  - Energía.
  - Astrología.
  - Servicio personalizado.
- Recomendación rápida según elección.
- CTA final para solicitar reserva.
- Recordatorio de que toda reserva necesita confirmación del administrador.
- Mejor lectura en móvil y PC.

## Flujo recomendado para cliente

1. Leer guía de servicios.
2. Elegir el tipo de servicio.
3. Revisar la sección de servicios.
4. Preparar mensaje si tiene dudas.
5. Solicitar reserva.
6. Esperar confirmación de Alaya.

## Importante

La solicitud de reserva no confirma la cita automáticamente.
El administrador debe confirmar la reserva.
