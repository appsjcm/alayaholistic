# Alaya Holistics v9.8 · Contacto y Confianza Pro

Esta versión mejora la sección de contacto y confianza.

## Mejoras

- Nueva guía antes de contacto.
- Bloques para:
  - Reservas.
  - Dudas.
  - Herbolario.
  - Talleres.
- Generador de mensaje de contacto.
- Botón para copiar mensaje.
- Bloque “Qué puedes esperar de Alaya”.
- CTA para preparar mensaje o ver ayuda.

## Mensaje de contacto

El cliente puede indicar:

- Nombre.
- Motivo.
- Contacto.
- Horario preferido.
- Detalles.

## Confianza

Se aclara que:

- Las reservas no son automáticas.
- Alaya confirma manualmente.
- Los textos holísticos y de herbolario son de bienestar general.
- Para temas de salud, debe consultarse con un profesional sanitario.
- Los textos legales son orientativos y deben adaptarse/revisarse antes de publicar.
