# Alaya Holistics v10.5 · Admin WordPress CMS Pro

Esta versión añade un panel Admin tipo WordPress.

## Qué controla

Desde la pestaña **WordPress CMS** se puede editar:

- Identidad de la web.
- Portada / Hero.
- Botones principales.
- Colores y escala visual.
- Menú destacado público.
- Bloques/secciones públicas.
- SEO básico.
- Aviso global.
- Estado de publicación.

## Funciones principales

### Identidad

- Nombre de la web.
- Subtítulo.
- Email.
- Teléfono.
- Ubicación.
- Instagram.
- Texto corto de marca.

### Portada

- Título principal.
- Texto principal.
- Botón principal.
- Botón secundario.

### Apariencia

- Color principal.
- Color secundario.
- Color acento.
- Escala visual.
- Modo de portada.

### Menú

Permite crear enlaces rápidos públicos.

### Bloques

Permite crear bloques públicos como:

- Destacados.
- Servicios.
- Talleres.
- Herbolario.
- Avisos.
- FAQ.

### SEO

Permite configurar:

- Title.
- Description.
- Keywords.
- Indexación.

### Publicar

Permite:

- Activar aviso global.
- Poner texto de aviso.
- Exportar JSON.
- Copiar JSON.
- Importar JSON.

## Importante

Esto no es WordPress real con PHP/MySQL.
Es un mini CMS tipo WordPress para hosting estático.

Los cambios se guardan en el navegador mediante localStorage.
Para que varios administradores o dispositivos compartan datos, hay que conectar Firebase/online sync.
