# Alaya Holistics v8.2 · Pre-reserva Guiada Cliente Pro

Esta versión añade una pre-reserva guiada para clientes.

## Nuevo

- Sección `#pre-reserva`.
- Campos públicos:
  - Nombre.
  - Motivo.
  - Preferencia de horario.
  - Canal de contacto.
  - Comentario opcional.
- Generación automática de mensaje.
- Botón copiar mensaje.
- Botón ir a reservas.
- Botón contactar.

## Objetivo

Que el cliente prepare una solicitud clara antes de escribir a Alaya o enviar una reserva.

## Importante

La pre-reserva no confirma automáticamente una cita. La confirmación depende de la disponibilidad de Alaya.
