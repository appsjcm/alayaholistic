# Alaya Holistics v8.3 · Consulta de Reserva Cliente Pro

Esta versión añade una consulta pública de estado de reserva para clientes.

## Nuevo

- Sección `#consulta-reserva`.
- Búsqueda por:
  - Código de reserva.
  - Email o teléfono usado al reservar.
- Resultado público:
  - Código.
  - Estado.
  - Servicio o taller.
  - Fecha.
  - Hora.
  - Último movimiento visible.
- Botón copiar resumen.
- Botón contactar con Alaya.

## Privacidad

La sección no muestra:

- Notas internas.
- Auditoría.
- Backups.
- Métricas.
- Herramientas de administrador.
- Información privada de otras reservas.

## Nota

En modo estático, consulta los datos guardados en el navegador/dispositivo donde exista la reserva.
Para consulta centralizada real se puede conectar a Firebase con reglas de seguridad adecuadas.
