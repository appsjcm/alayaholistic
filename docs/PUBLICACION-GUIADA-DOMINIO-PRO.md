# Alaya Holistics v12.1 · Publicación Guiada y Dominio Pro

Esta versión añade un asistente final de publicación.

## Plataformas

- GitHub Pages.
- Netlify.
- Vercel.
- Hosting manual.

## Qué permite

- Guardar URL pública.
- Guardar ruta base.
- Guardar dominio personalizado.
- Guardar email de contacto.
- Generar pasos de publicación.
- Generar `robots.txt`.
- Generar `sitemap.xml`.
- Exportar plan de publicación JSON.
- Copiar pasos de subida.

## Flujo recomendado

1. Entrar en Admin → Backup.
2. Descargar backup total.
3. Entrar en Admin → Legal Pro.
4. Revisar textos.
5. Entrar en Admin → SEO UX.
6. Ejecutar auditoría.
7. Entrar en Admin → Lanzamiento QA.
8. Ejecutar QA.
9. Entrar en Admin → Release Final.
10. Generar informe final.
11. Entrar en Admin → Publicación.
12. Elegir plataforma y generar plan.
13. Subir archivos al hosting.
