# Alaya Holistics v10.2 · UX Global Pro

Esta versión mejora la experiencia general de la web.

## Añadido

- Barra superior de progreso.
- Botón flotante para volver arriba.
- Mapa rápido de la web.
- Enlace “Saltar al contenido”.
- Foco visual más claro para accesibilidad.
- Mejoras de lectura, contraste y navegación.
- Mejoras ligeras en páginas 404/offline.

## Objetivo

Que la web se sienta más cómoda, profesional y fácil de navegar en móvil y PC.

## Separación

El mapa rápido incluye acceso a Admin, pero el panel interno sigue separado en `admin.html`.
