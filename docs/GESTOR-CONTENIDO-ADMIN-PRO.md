# Alaya Holistics v10.3 · Gestor de Contenido Admin Pro

Esta versión añade un gestor de contenido dentro del panel Admin.

## Qué permite crear

Desde Admin se puede añadir:

- Servicios.
- Talleres.
- Cursos.
- Productos de herbolario.
- Avisos.
- Novedades.

## Acciones disponibles

- Crear contenido.
- Editar contenido.
- Activar o desactivar visibilidad.
- Duplicar contenido.
- Borrar contenido.
- Exportar contenido en JSON.

## Dónde aparece

El contenido activo aparece automáticamente en:

- Servicios.
- Talleres y Cursos.
- Herbolario.
- Avisos y novedades.

## Uso recomendado

1. Entrar al panel Admin.
2. Abrir la pestaña “Gestor contenido”.
3. Elegir tipo de contenido.
4. Completar título, categoría, estado, fecha, precio y descripción.
5. Marcar “Publicar visible en la web”.
6. Guardar.

## Importante

En esta versión estática el contenido se guarda en el navegador con localStorage.
Para compartir contenido entre dispositivos o administradores hace falta activar sincronización online/Firebase.
