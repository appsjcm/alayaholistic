# Alaya Holistics v11.6 · Backup y Restauración Pro

Esta versión añade un centro de copias de seguridad en Admin.

## Qué permite

- Exportar backup total.
- Exportar solo partes seleccionadas.
- Restaurar backup JSON.
- Combinar datos con los actuales.
- Reemplazar datos actuales.
- Auditar datos guardados en el navegador.
- Copiar resumen de backup.
- Preparar migraciones entre dispositivos.

## Datos incluidos

- CMS / WordPress interno.
- Reservas online.
- Agenda y bloqueos.
- Clientes CRM.
- Mensajes.
- Configuración Sync/Firebase.

## Uso recomendado

Antes de actualizar la web:

1. Entrar en Admin → Backup.
2. Descargar backup total.
3. Guardar el ZIP de la versión actual.
4. Subir la nueva versión.
5. Restaurar backup si cambias de navegador o dispositivo.
6. Revisar reservas, agenda, clientes y CMS.

## Privacidad

Los backups pueden contener datos de clientes.
No los publiques ni los compartas en enlaces abiertos.
