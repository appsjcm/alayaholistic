# Alaya Holistics v6.1 · Contenido Público Pro

Esta versión añade un editor de contenido público para testimonios y preguntas frecuentes.

## Nuevo

- Nueva pestaña admin **Contenido**.
- Editor de testimonios/opiniones.
- Editor de preguntas frecuentes.
- Sección pública **Opiniones y dudas**.
- Contadores públicos de opiniones y FAQs.
- Botones admin:
  - añadir testimonio
  - cargar ejemplos
  - añadir FAQ
  - copiar contenido JSON
  - restaurar contenido base
- Edición rápida: editar mueve el contenido al formulario y lo quita de la lista para guardarlo actualizado.
- Sitemap actualizado con `#opiniones-faq`.

## Uso recomendado

1. Entrar al admin.
2. Abrir pestaña **Contenido**.
3. Añadir testimonios reales cuando existan.
4. Añadir FAQs según dudas frecuentes de clientes.
5. Revisar la sección pública **Opiniones y dudas**.

## Nota

Los testimonios de ejemplo son contenido base para diseño. Sustituir por testimonios reales antes de publicar si se desea.
