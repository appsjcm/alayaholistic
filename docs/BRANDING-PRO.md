# Alaya Holistics v3.9 · Branding Pro

Esta versión permite personalizar la identidad visual desde el panel admin.

## Nuevo

En:

```txt
Panel admin > Ajustes > Branding Pro
```

puedes editar:

- estilo visual
- color principal
- color secundario
- color luz
- texto superior del hero
- etiqueta de agenda
- descripción principal
- textos de botones
- badges de la portada
- título y texto de la tarjeta principal

## Estilos incluidos

- Cosmic premium
- Herbal natural
- Sunrise cálido
- Ocean calm
- Personalizado

## Cómo usarlo

1. Entra en el panel admin.
2. Abre Ajustes.
3. Edita Branding Pro.
4. Pulsa **Aplicar vista previa**.
5. Pulsa **Guardar ajustes** para conservarlo.

## Importante

La vista previa cambia la portada al momento, pero para que quede guardado debes pulsar **Guardar ajustes**.

Si usas Firebase, después puedes sincronizar ajustes para compartir el branding entre dispositivos.
