# Alaya Holistics v5.2 · Privacidad Astral Pro

Esta versión refuerza el área interna de cartas astrales.

## Nuevo

- Panel **Privacidad Astral Pro**.
- Modo privacidad para ocultar visualmente notas privadas y seguimiento.
- Plantilla de consentimiento copiable.
- Checklist de privacidad copiable.
- Fecha de consentimiento.
- Origen del consentimiento.
- Fecha de revisión de datos.
- Etiquetas internas de datos.
- Notas de consentimiento.
- Botón para anonimizar carta astral.
- Indicadores visuales de consentimiento en cada ficha.

## Modo privacidad

El modo privacidad difumina notas privadas, seguimiento y sesiones en pantalla.
Al pasar el cursor por encima se puede revisar el contenido.

## Anonimización

La anonimización elimina datos de contacto y notas privadas identificables.
No sustituye una política de protección de datos completa.

## Recomendación

Antes de usar en producción, revisar protección de datos, textos legales y consentimiento con asesoramiento profesional.
