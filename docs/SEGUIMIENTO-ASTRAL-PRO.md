# Alaya Holistics v5.4 · Seguimiento Astral Pro

Esta versión añade seguimiento interno de clientes/pacientes dentro del módulo de cartas astrales.

## Nuevo

- Fecha de próxima sesión / seguimiento.
- Prioridad:
  - Alta
  - Normal
  - Baja
- Próximo paso concreto.
- Panel **Seguimiento Astral Pro**.
- Filtros:
  - Todos
  - Vencidos
  - Hoy
  - Próximos 7 días
  - Prioridad alta
- Botón para abrir ficha.
- Botón para copiar mensaje de seguimiento.
- Botón para marcar seguimiento como hecho.
- Informes imprimibles incluyen próximo paso.
- El Client Hub muestra el próximo paso más útil.

## Uso recomendado

1. Al finalizar una consulta, anotar el próximo paso.
2. Asignar fecha y prioridad.
3. Revisar cada día el panel Seguimiento Astral Pro.
4. Copiar mensaje de seguimiento si se quiere contactar.
5. Marcar como hecho cuando ya esté gestionado.

## Privacidad

Los seguimientos pueden incluir datos personales. Revisar siempre el contenido antes de copiar o enviar.
