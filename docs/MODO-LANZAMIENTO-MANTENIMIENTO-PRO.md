# Alaya Holistics v7.4 · Modo Lanzamiento y Mantenimiento Pro

Esta versión añade un modo especial para indicar que la web está en preparación, mantenimiento o lanzamiento.

## Nuevo

- Nueva pestaña admin **Mantenimiento**.
- Activar/desactivar modo especial.
- Modos:
  - Próximamente
  - Mantenimiento
  - Lanzamiento
- Visualización:
  - solo tarjeta pública
  - pantalla completa
  - pantalla completa + tarjeta
- Título editable.
- Mensaje editable.
- Fecha objetivo con cuenta atrás.
- Botón CTA editable.
- Opción para permitir entrar igualmente.
- Vista previa en admin.
- Pantalla pública opcional.
- Sección pública `#estado-alaya`.
- Botón copiar estado.
- Botón copiar snippet.
- Botón copiar checklist.
- Sitemap actualizado.

## Uso recomendado

- Antes de publicar la web.
- Durante mantenimiento.
- Para anunciar una fecha de lanzamiento.
- Para campañas o aperturas especiales.

## Nota

El modo se guarda localmente en el navegador. Para hacerlo global en una web estática,
copia el snippet o deja el HTML fijo antes de publicar.
