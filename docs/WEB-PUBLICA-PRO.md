# Alaya Holistics v5.6 · Web Pública Pro

Esta versión mejora la parte pública de la web.

## Nuevo

- SEO básico en el `<head>`.
- Datos estructurados JSON-LD tipo LocalBusiness.
- Open Graph para compartir.
- Twitter Card.
- Theme color.
- `sitemap.xml`.
- `robots.txt`.
- `offline.html`.
- Fallback offline para PWA.
- Accesos rápidos flotantes:
  - Reservar
  - Servicios
  - Herbolario
  - Contacto
- Enlace accesible para saltar al contenido.
- Nueva sección **Experiencia web**.
- Mapa rápido en el footer.
- Manifest PWA mejorado.

## Objetivo

Que la web pública sea más clara, navegable y preparada para crecer antes del lanzamiento.

## Nota

Para SEO final, cuando se publique la web en dominio real, conviene actualizar URLs absolutas, nombre legal, dirección si procede y enlaces sociales reales.
