# Alaya Holistics v9.9 · Admin Visible Pro

Esta versión vuelve a mostrar un botón de acceso al administrador en la página principal.

## Añadido

- Botón **Admin** visible en el menú principal.
- Bloque público **Acceso administrador**.
- Enlace a `admin.html`.
- Enlace al `manual-admin.html`.
- Acceso también desde el footer.
- Mejoras visuales en móvil y PC.

## Separación

Aunque el botón sea visible, la separación se mantiene:

- Clientes: `index.html`
- Administración: `admin.html`
- Manual interno: `manual-admin.html`

## Recomendación

El acceso visible facilita la entrada del administrador, pero no sustituye medidas reales de seguridad.
Las credenciales y permisos deben configurarse correctamente en producción.
