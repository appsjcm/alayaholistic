# Alaya Holistics v5.7 · Centro de Lanzamiento Pro

Esta versión añade un panel admin para preparar la publicación de la web.

## Nuevo

- Pestaña admin **Lanzamiento**.
- Puntuación de preparación de lanzamiento.
- Checklist público:
  - logos
  - textos
  - contacto
  - SEO
  - móvil
  - PWA/offline
- Checklist admin:
  - login
  - roles
  - reservas
  - Firebase
  - privacidad
  - backup
- Pruebas rápidas automáticas.
- Informe de lanzamiento copiable.
- Notas internas de lanzamiento.
- Bloque público “Web preparada”.
- Mejoras de carga de imágenes con `loading` y `decoding`.
- Manifest PWA revisado.

## Uso recomendado

1. Entrar al panel admin.
2. Abrir la pestaña **Lanzamiento**.
3. Pulsar **Revisar ahora**.
4. Marcar puntos revisados.
5. Copiar informe de lanzamiento.
6. Exportar backup antes de publicar.

## Importante

Este centro ayuda a revisar, pero antes de publicar conviene probar en móvil real y revisar Firebase/privacidad.
