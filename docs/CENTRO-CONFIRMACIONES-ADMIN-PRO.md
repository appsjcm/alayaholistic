# Alaya Holistics v8.5 · Centro de Confirmaciones Admin Pro

Esta versión añade una bandeja interna para confirmar reservas manualmente.

## Nuevo

- Pestaña admin **Confirmaciones**.
- Contadores de:
  - Pendientes.
  - Confirmadas.
  - Canceladas.
- Filtro por estado.
- Buscador por cliente, código, servicio o contacto.
- Botón **Confirmar y copiar mensaje**.
- Botón **Cancelar y copiar mensaje**.
- Botón **Copiar revisión**.
- Botón **Copiar informe pendientes**.
- Botón **Copiar protocolo**.

## Flujo correcto

1. El cliente envía una solicitud.
2. La solicitud queda pendiente.
3. El administrador entra en `admin.html`.
4. Abre la pestaña **Confirmaciones**.
5. Revisa disponibilidad.
6. Pulsa **Confirmar y copiar mensaje**.
7. Envía el mensaje al cliente.

## Importante

Una reserva pendiente no es una cita confirmada.
Solo queda confirmada cuando el administrador la confirma manualmente.
