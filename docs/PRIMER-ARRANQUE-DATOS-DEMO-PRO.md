# Alaya Holistics v12.3 · Primer Arranque y Datos Demo Pro

Esta versión añade un asistente para empezar a usar la web desde cero.

## Qué permite

- Guardar datos básicos del proyecto.
- Marcar estado inicial.
- Cargar datos demo.
- Borrar solo datos demo.
- Revisar checklist inicial.
- Ver prioridades.
- Exportar plan de primer arranque JSON.

## Datos demo

Crea ejemplos marcados con `demo-v12.3`:

- reserva demo,
- cliente demo,
- horario demo,
- mensaje demo.

El botón “Borrar demo” elimina solo esos elementos, no datos reales.

## Uso recomendado

1. Entrar en Admin → Primer Arranque.
2. Completar datos básicos.
3. Cargar demo.
4. Probar reserva, agenda, CRM y mensajes.
5. Borrar demo.
6. Descargar backup total.
7. Empezar con datos reales.
