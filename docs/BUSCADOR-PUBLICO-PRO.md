# Alaya Holistics v6.3 · Buscador Público Pro

Esta versión añade un buscador global para la web pública.

## Nuevo

- Nueva sección **Buscar en Alaya**.
- Buscador global público.
- Etiquetas rápidas:
  - Tarot
  - Reiki
  - Carta astral
  - Herbolario
  - Taller
  - Reserva
- Resultados de:
  - servicios
  - talleres/cursos
  - herbolario
  - novedades/blog
  - preguntas frecuentes
  - opiniones
  - contacto
  - reservas
  - astrología
- Estado vacío con CTA a reservar.
- Atajo de teclado `/` para ir al buscador.
- Sitemap actualizado con `#buscar-alaya`.

## Objetivo

Facilitar que el cliente encuentre rápido lo que necesita sin navegar por toda la web.

## Nota

El buscador funciona en navegador con los datos cargados en la propia web y el contenido guardado en localStorage.
