# Alaya Holistics v5.0 · Astral Admin Plus

Esta versión mejora el módulo interno de cartas astrales.

## Nuevo

- Historial de sesiones por carta astral.
- Notas por sesión.
- Próximo paso recomendado.
- Campos estructurados:
  - objetivo de consulta
  - fortalezas
  - retos
  - recomendaciones
- Botón para copiar prompt IA.
- Botón para insertar plantilla de interpretación.
- Vista resumida de secciones en cada ficha.
- Timeline de sesiones.
- Exportación JSON/CSV ampliada.

## Uso recomendado

1. Crear ficha con datos natales.
2. Abrir la herramienta Astral IA desde admin.
3. Copiar el prompt IA desde la ficha.
4. Pegar el resultado en interpretación, fortalezas, retos y recomendaciones.
5. Guardar notas privadas y sesiones de seguimiento.

## Privacidad

Este módulo guarda datos personales y notas internas.
Debe usarse con consentimiento y revisar la privacidad antes de producción.
