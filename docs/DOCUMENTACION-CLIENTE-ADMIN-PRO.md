# Alaya Holistics v7.8 · Documentación Cliente/Admin Pro

Esta versión separa la documentación pública de clientes y el manual interno de administración.

## Nuevo

- `guia-clientes.html`
  - Página pública.
  - Pensada para clientes.
  - Explica reservas, servicios, herbolario, talleres, carta astral, contacto e instalación como app.

- `manual-admin.html`
  - Página interna.
  - Marcada como `noindex,nofollow`.
  - Bloqueada en `robots.txt`.
  - Explica reservas internas, servicios, herbolario, talleres, cartas astrales, backups, SEO, calidad y publicación.

## Objetivo

Evitar que un cliente vea instrucciones técnicas o internas del administrador.

## Recomendación

- Enviar a clientes solo `guia-clientes.html`.
- Usar `manual-admin.html` solo para administración.
- Mantener el panel interno separado de la web pública.
