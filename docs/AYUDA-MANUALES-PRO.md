# Alaya Holistics v10.1 · Ayuda y Manuales Pro

Esta versión mejora la ayuda pública y los manuales.

## Añadido

- Sección pública “Ayuda rápida”.
- FAQ para clientes.
- Botón para copiar pasos rápidos.
- Mejoras en `guia-clientes.html`.
- Mejoras en `manual-admin.html`.

## Ayuda para clientes

Explica cómo:

- Elegir servicio.
- Solicitar reserva.
- Consultar estado.
- Responder propuesta.
- Contactar con Alaya.
- Entender que la confirmación no es automática.

## Manual admin

Añade rutina interna para:

- Revisar pendientes.
- Confirmar o proponer alternativas.
- Actualizar contenido.
- Revisar que no se mezclan herramientas internas con cliente.
- Hacer backup tras cambios importantes.

## Separación

Cliente: ayuda pública y guía clientes.  
Admin: manual interno y panel administrador.
