# Alaya Holistics v12.7 · Multiidioma y Traducciones Pro

Esta versión añade un centro de traducciones y un selector público de idioma.

## Idiomas incluidos

- Castellano.
- Català.
- English.

## Qué permite

- Cambiar idioma en la web pública.
- Editar textos clave.
- Guardar diccionario.
- Exportar traducciones JSON.
- Importar traducciones JSON.
- Copiar resumen de textos.

## Uso recomendado

1. Entrar en Admin → Idiomas.
2. Elegir idioma.
3. Revisar textos.
4. Guardar traducciones.
5. Aplicar idioma.
6. Exportar JSON como copia.

## Nota

El sistema traduce los textos clave preparados.
Para traducir absolutamente toda la web, se puede ampliar el diccionario con nuevas claves por sección.
