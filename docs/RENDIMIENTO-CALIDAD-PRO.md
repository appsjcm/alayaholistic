# Alaya Holistics v7.2 · Rendimiento y Calidad Pro

Esta versión añade una auditoría técnica local para revisar la web antes de publicar.

## Nuevo

- Nueva pestaña admin **Calidad**.
- Puntuación global de calidad.
- Revisión de archivos críticos.
- Revisión de enlaces internos.
- Revisión de imágenes sin `alt`.
- Revisión de imágenes sin `loading`.
- Revisión básica de formularios.
- Revisión básica de SEO.
- Informe copiable.
- Exportación CSV.
- Checklist de calidad.
- Sección pública de calidad técnica.
- Sitemap actualizado con `#calidad-alaya`.

## Uso recomendado

1. Entrar en admin.
2. Abrir pestaña **Calidad**.
3. Pulsar **Ejecutar auditoría**.
4. Revisar avisos.
5. Exportar CSV si se quiere conservar el reporte.
6. Probar también en móvil real antes de publicar.

## Nota

La auditoría es local y orientativa. No sustituye pruebas avanzadas de rendimiento en herramientas externas.
