# Alaya Holistics v8.7 · Respuesta a Propuesta Cliente Pro

Esta versión completa el flujo de horarios alternativos.

## Cliente

Nueva sección pública `#responder-propuesta`:

- Introduce código de solicitud.
- Introduce email o teléfono.
- Si hay alternativas, elige una.
- Copia un mensaje aceptando la opción.
- Envía el mensaje a Alaya.

## Administrador

Dentro de la pestaña **Alternativas**:

- Selecciona una reserva con alternativas.
- Selecciona la opción aceptada por el cliente.
- Registra la aceptación.
- Confirma manualmente la reserva.
- Copia el mensaje final de confirmación.

## Flujo completo

1. Cliente solicita reserva.
2. Admin propone alternativas.
3. Cliente acepta una opción.
4. Admin registra la aceptación.
5. Admin confirma manualmente la reserva.

## Importante

Aceptar una alternativa no confirma automáticamente la cita.
La confirmación final siempre la hace el administrador.
