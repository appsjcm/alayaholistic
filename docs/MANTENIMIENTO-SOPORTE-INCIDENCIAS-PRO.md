# Alaya Holistics v12.4 · Mantenimiento, Soporte e Incidencias Pro

Esta versión añade un centro de mantenimiento en Admin.

## Qué permite

- Definir estado público:
  - funcionando,
  - en revisión,
  - mantenimiento.
- Registrar incidencias.
- Editar incidencias.
- Cambiar estado a en proceso o cerrada.
- Borrar incidencias cerradas.
- Ver auditoría rápida de módulos críticos.
- Copiar resumen de soporte.
- Exportar reporte JSON.

## Uso recomendado

1. Entrar en Admin → Mantenimiento.
2. Registrar errores o mejoras.
3. Priorizar incidencias.
4. Cerrar incidencias resueltas.
5. Exportar reporte antes de cambios grandes.
6. Usar Backup antes de actualizar la web.

## Después de publicar

Usa este panel para organizar mantenimiento, tareas pendientes y mejoras futuras.
