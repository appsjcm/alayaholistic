# Alaya Holistics v9.0 · Diseño Responsive Total Pro

Esta versión ajusta la web para móvil, tablet y PC.

## Mejoras en móvil

- Menú desplegable más cómodo.
- Enlaces ordenados en columnas.
- Barra inferior móvil con accesos rápidos:
  - Inicio.
  - Servicios.
  - Reservar.
  - Estado.
  - Contacto.
- Botones más grandes.
- Formularios a una columna.
- Tarjetas a una columna.
- Textos y títulos adaptados.
- Panel admin con pestañas horizontales desplazables.

## Mejoras en PC

- Se mantiene el menú superior limpio.
- Ancho amplio para secciones.
- Grids más equilibrados.
- Hero y panel inicial mejor organizados.
- Panel administrador más cómodo.

## Separación cliente/admin

- Cliente: `index.html`
- Administrador: `admin.html`

La barra móvil es solo para la zona pública de clientes.
