# Alaya Holistics v4.7 · Diseño Astral Global

Esta versión aplica el estilo visual de **Alaya Astral IA** a toda la web principal.

## Cambios principales

- Fondo cósmico oscuro inspirado en Alaya Astral.
- Capa de estrellas animada.
- Paleta astral global: violeta, dorado, cyan, rosa astral y crema cálido.
- Tarjetas tipo glass / cristal astral.
- Botones con gradientes astrales.
- Header más oscuro y premium.
- Servicios, talleres, herbolario y astrología con estética unificada.
- Nuevo preset de branding: **Alaya Astral**.
- Sello astral en el hero principal.
- Nueva sección visual **Universo Alaya**.

## Objetivo

Que toda la web tenga el mismo estilo visual que la app Alaya Astral IA,
manteniendo las secciones separadas:

- Alaya Holistics
- Alaya Herbolario
- Alaya Astral IA
