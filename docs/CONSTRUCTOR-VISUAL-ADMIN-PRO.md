# Alaya Holistics v10.6 · Constructor Visual Admin Pro

Esta versión añade un constructor visual dentro del Admin.

## Qué permite

- Crear bloques por plantilla.
- Ordenar bloques.
- Duplicar bloques.
- Publicar u ocultar bloques.
- Borrar bloques.
- Guardar revisiones.
- Restaurar revisiones.
- Exportar backup completo.

## Plantillas incluidas

- Hero promocional.
- Servicio destacado.
- Taller o curso.
- Herbolario.
- Aviso importante.
- Pregunta frecuente.
- Llamada a la acción.

## Flujo recomendado

1. Abrir Admin.
2. Entrar en “Constructor visual”.
3. Guardar una revisión.
4. Añadir una plantilla.
5. Editar el bloque desde “WordPress CMS → Bloques”.
6. Ordenar los bloques.
7. Revisar en la web pública.
8. Exportar backup JSON.

## Nota

En hosting estático, los cambios se guardan en el navegador.
Para multiadministrador y sincronización real, conectar Firebase/online sync.
